# 🤖 內部知識助理 RAG 專案

以 LangChain + 本地模型建構的 Retrieval-Augmented Generation (RAG) 問答系統，支援企業內部知識文件查詢，包含關鍵詞抽取、多查詢檢索、Hybrid Search、重排序與本地 LLM 回答模組。

## 📌 專案特色

- ✅ 支援 PDF 文件切段、標題結構對應
- 🧠 關鍵詞抽取 + 多查詢擴增提升召回率
- 🔍 Hybrid Search（向量 + BM25）結合語意與關鍵詞檢索
- 🔝 Reranker 使用 BGE 強化文段排序
- 🔒 支援本地部署，保有企業資料隱私
